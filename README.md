# lead-scoring-eda

Exploratory data analysis on X Education's lead scoring dataset.

## What's inside
- Conversion rate breakdowns by lead source, occupation, geography, and tags
- Communication opt-out impact analysis
- Numeric feature correlations with conversion
- Marketing insights and recommendations throughout

## Dataset
[X Education Lead Scoring – Kaggle](https://www.kaggle.com/datasets/amritachatterjee09/lead-scoring-dataset)

## Stack
Python · Pandas · Matplotlib · Seaborn · Jupyter Notebook
